Questa sezione fornisce tutorial passo-passo per costruire con Goa-AI, da agenti semplici a pattern di composizione avanzata:

- **Agente Semplice** – Crea un agente base e impara le basi dell'uso dei tool
- **Composizione Agenti** – Costruisci sistemi multi-agente dove gli agenti chiamano altri agenti come tool
- **Toolset MCP** – Integra con servizi esterni usando Model Context Protocol
